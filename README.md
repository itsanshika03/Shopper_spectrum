# 🛍️ Shopper Spectrum

An end-to-end Customer Analytics & Recommendation System developed using Python, Streamlit, SQL, Machine Learning and Data Visualization.

---

## 🌐 Live Demo

👉 https://shopper-spectrum-dashboard.streamlit.app/

---

## 📌 Project Overview

Shopper Spectrum helps businesses understand customer purchasing behaviour through interactive dashboards, customer segmentation, product recommendations and business analytics.

The application combines machine learning models with visual analytics to support data-driven business decisions.

---

## 🚀 Features

- 📊 Interactive Business Dashboard
- 👥 Customer Segmentation (K-Means Clustering)
- 🛒 Product Recommendation System
- 📈 Sales Analytics
- 📉 Customer Insights
- 🗄 SQL Business Queries
- ✏ CRUD Operations
- 🎨 Professional Streamlit UI
